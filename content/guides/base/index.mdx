<Callout title="Who is this for?">
This guide is for participants in the **Based Off** category of Boot YSWS — where you create a custom OS using an existing OS as the foundation. Ubuntu 24.04 LTS is a great starting point.
</Callout>

These guides will walk you through:

1. **[Setting up your development environment](/guides/base/setup)** — install QEMU, get the Ubuntu 24.04 Server ISO, and prepare your tools
2. **[Customizing with Cubic](/guides/base/cubic)** — the easiest GUI-based path to a custom ISO
3. **[Customizing with live-build](/guides/base/live-build)** — a fully reproducible CLI approach
4. **[Testing your custom ISO](/guides/base/testing)** — boot your image in QEMU and verify the Based Off requirements
